| SPM dependency supply chain | `KeyboardShortcuts` source is fetched from GitHub at resolve time. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Spoofing | Global shortcut registration | accept | First-registered-hotkey-wins is an OS-level behavior; a malicious app claiming the same combo is not mitigable at the app layer. Local single-user utility; low value target. |
| T-02-02 | Elevation of Privilege | Global key monitoring | accept | KeyboardShortcuts uses Carbon `RegisterEventHotKey`, which does NOT require Accessibility/Input Monitoring permission and cannot read arbitrary keystrokes — it only receives the registered combo. No keystroke logging surface introduced. |
| T-02-SC | Tampering | SPM install of KeyboardShortcuts | mitigate | Package Legitimacy Audit in 02-RESEARCH.md verified the package via Swift Package Index + GitHub (verdict OK, author Sindre Sorhus, high reputation, used in production apps). Version pinned `from: "3.0.1"`; `Package.resolved` commits the resolved revision. No `[ASSUMED]`/`[SUS]` packages, so no blocking checkpoint required. |
