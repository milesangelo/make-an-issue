<objective>
Vendor the speech-to-text toolchain into the app bundle so transcription is zero-config: a new `scripts/fetch-whisper.sh` builds `whisper-cli` from source at the pinned git tag `v1.9.1` and downloads the SHA256-verified `ggml-small.en.bin` model into a gitignored `vendor/` directory, and `scripts/build-app.sh` is extended to copy both artifacts into `MakeAnIssue.app/Contents/Resources` and ad-hoc sign the binary. This delivers the binary + model half of TRANSCRIBE-01 (the rewire half is plan 03-04).

Purpose: Removes user setup friction — the user no longer supplies an ASR command; the binary and model ride inside the `.app`. Keeps the ~466 MB artifacts out of git while making them reproducible via pinned tag + checksum.
Output: New `scripts/fetch-whisper.sh`; extended `scripts/build-app.sh`; `vendor/` added to `.gitignore`.
</objective>
