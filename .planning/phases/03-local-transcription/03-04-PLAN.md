## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| bundled binary/model/wav paths → `/bin/zsh -lc` command (via CLIRunner) | Three filesystem paths are interpolated into a shell command string before reaching the shell. |
| whisper-cli stdout → AppState.transcript → MenuView | External-process output crosses back and is stored + rendered as selectable text. |
| Bundle.main.resourceURL resolution → spawned subprocess | The app trusts the bundle path to locate the executable it spawns. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-13 | Tampering | path interpolation in `Transcriber.run(wavURL:resourceBase:)` | mitigate | All three paths (binary, model, wav) are wrapped in POSIX single quotes with the `'\\''` close/literal/reopen escape (ASVS V5), so a path with spaces or shell metacharacters cannot break out of its argument or trigger expansion. Verified by `testRunConstructsCorrectCommand`. |
